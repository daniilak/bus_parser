# Парсер транспорта

Этот проект представляет собой парсер данных о транспорте, который собирает информацию о местоположении транспортных средств и сохраняет её в базу данных.

## Описание

Парсер работает в непрерывном режиме, собирая данные о местоположении транспортных средств. Он имеет следующие особенности:

- Автоматическое сохранение данных в базу данных
- Обработка ошибок и автоматические повторные попытки
- Различные интервалы обновления в зависимости от времени суток
- Логирование всех операций
- Поддержка московского часового пояса

## Установка

1. Клонируйте репозиторий
2. Установите зависимости:
```bash
pip install -r requirements.txt
```

## Использование

Для запуска парсера выполните:
```bash
python main.py
```

## Структура проекта

- `main.py` - основной файл парсера
- `lib.py` - библиотека с функциями парсинга
- `models.py` - модели базы данных
- `requirements.txt` - зависимости проекта

## Особенности работы

- В обычное время парсер обновляет данные каждую минуту
- При возникновении ошибок интервал обновления сокращается до 10 секунд
- В ночное время (0:00 - 4:00 МСК) интервал обновления увеличивается до 20 минут
- Все операции логируются в файл `transport_parser.log`

## Требования

- Python 3.7+
- Доступ к интернету
- База данных (настраивается в `models.py`)
